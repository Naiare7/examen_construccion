# Examen Construcción

Este repositorio contiene el código fuente desarrollado para el examen práctico de construcción web. El proyecto consiste en el diseño y maquetación de una página web estática con varias secciones enlazadas entre sí.

## Descripción del proyecto

La página web está estructurada en múltiples documentos HTML para separar el contenido, incluyendo:
- Página de inicio
- Sección de servicios
- Catálogo de productos
- Información de patrocinios
- Formulario de contacto

Todo el diseño visual y la estructura responsive se han implementado utilizando CSS, organizadas en la carpeta css.

## Tecnologías utilizadas

- HTML5 para la estructura semántica.
- CSS3 para el diseño, maquetación y el responsive para diferentes dispositivos.

## Ejecución

Al ser un proyecto estático, no requiere de la instalación de dependencias ni servidores locales. Para probar la web, basta con descargar los archivos y abrir el documento `index.html` en cualquier navegador moderno.
